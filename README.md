# jiant-for-VL
これはjiantをベースにVisualBERTとLXMERT、CLIPの言語的知識を分析できるようにしたレポジトリです。

## Getting Started
言語知識に関するデータセットは./probing/dataのスクリプトから取得してください。ただし。データセットは一部有料のため、本レポジトリでは共有できません。


## Running
depok.ymlと同じ環境を用意してください。
